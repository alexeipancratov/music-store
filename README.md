# Music Store web app

## Technologies stack
* Node.js + Express
* React
* Bootstrap
* HTML5

## How to run the project
1. npm install in the root directory
2. cd /server -> npm install
3. cd /ui -> npm install
4. npm start from the root folder

Default admin application user:
  - admin/admin
